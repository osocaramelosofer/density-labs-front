## Run Locally


node version v18.18.2

Install density-labs-front with npm

Clone the project

```bash
  git clone https://github.com/osocaramelosofer/density-labs-front
```

Go to the project directory

```bash
  cd density-labs-front
```

Install dependencies with pnpm

```bash
  npm install
```

Start the server

```bash
  pnpm run dev
```

Run the test

```bash
  npm run test
```
